LA GOURMETISEMON PROJET

Application web

La Gourmetise propose une application web permettant aux boulangeries de s'inscrire au concours et aux visiteurs de découvrir les participants. Elle offre une expérience utilisateur fluide sur tous les appareils.

Application mobile

Destinée aux juges, l'application mobile facilite la notation des boulangeries avec une interface intuitive et une fonctionnalité hors ligne grâce à une base de données embarquée.

Méthode agile

L'utilisation de sprints et de Trello structure la gestion du projet, favorisant une approche Agile qui permet d'ajuster et d'évaluer le développement en continu.

Outils de développement et de versionning

GitHub soutient le développement collaboratif de l'application, avec des branches dédiées aux nouvelles fonctionnalités pour une intégration harmonieuse.

Base de données

Le projet intègre une base de données centralisée pour l'application web et une version embarquée pour l'application mobile, optimisant la gestion des données.

API

Une API, testée avec Postman, assure la synchronisation entre l'application mobile et la base de données centralisée, facilitant le partage et la mise à jour des informations.
